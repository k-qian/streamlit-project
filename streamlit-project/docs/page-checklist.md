# 媽媽健康日誌 — 頁面交付清單

## 登入前

- 登入 — 帳號密碼登入
- 註冊 — 建立新帳號（與登入同頁，分頁籤）

## 登入後（側邊欄）

- 總覽 — 歡迎頁、指標卡片、快速操作入口
- 健康日誌 — 每日生理數據輸入表單 + 歷史記錄列表 + 備註部分需要ai情緒分析
- 數據儀錶板 — 體重/心率趨勢圖表
- 營養建議 — 依週數取得飲食建議 + 嵌入youtube影片
- AI 風險預警 — AI 健康風險分析
- 健康報告 — 選擇期間生成報告
- 通知中心 — 系統通知列表
- 托嬰中心查找 — 托嬰中心名單查詢（讀取 托嬰中心.xlsx）
- 登出 — 清除登入狀態

## 注意事項

- 本專案使用 Streamlit 框架開發，UI 元件由框架提供，無法像自由撰寫 HTML + CSS 一樣精細調整樣式與排版，視覺呈現會與原有設計稿有落差
- AI 功能使用 Gemini 3.1 Flash Lite 模型進行開發
